Auditor Document Requests

How this tab works

What this tab is for. The Planning and Execution tabs cover everything StudyTrust prepared in advance. But once TPCSA is in the room, they will almost always need something that was not on the original checklist. This tab is where those additional requests land and are tracked from start to resolution.

The workflow: Open → Assign (In Progress) → Prepare & Upload → Resolved (Teams notification fires) → Closed.

Personnel Documents

🔒 Restricted access. This tab is visible only to members of Dashboards - HR-Authorised-Personnel. Rows marked Internal Only are not visible to auditors. Change to Grant Auditor Access only for a specific audit request, then revert to Internal Only once closed.

Auditor Access — How it works. When TPCSA requests a personnel document, change the relevant row to Grant Auditor Access. Once the auditor confirms receipt and the request is Closed, revert to Internal Only and notify Simon Letsela to revoke folder access.
